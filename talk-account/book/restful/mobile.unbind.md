# 解绑手机号

> POST /v1/mobile/unbind

| 参数            | 类型               | 是否必须  | 描述  |
| -------------- | ------------------ | -------- | ------------ |
| phoneNumber    | String             | true     | 手机号码，国外手机号必须带区号信息 |

## 响应

```json
{
  "_id": "55ee455204b97fa8d3f25cfe",
  "__v": 0,
  "updatedAt": "2015-09-08T02:17:54.572Z",
  "createdAt": "2015-09-08T02:17:54.330Z",
  "wasNew": false,
  "id": "55ee455204b97fa8d3f25cfe"
}
```
