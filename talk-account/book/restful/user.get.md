# 读取账号信息

> GET /v1/user/get

| 参数            | 类型               | 是否必须  | 描述  |
| -------------- | ------------------ | -------- | ------------ |
| accountToken   | String             | true     | 用户口令 |

## 响应

```json
{
  "_id": "55e54fd0ab963e7d7071eba7",
  "name": "18500000000",
  "phoneNumber": "18500000000",
  "__v": 0,
  "updatedAt": "2015-09-01T07:12:16.459Z",
  "createdAt": "2015-09-01T07:12:16.458Z"
}
```
